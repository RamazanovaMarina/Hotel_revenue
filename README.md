# Hotel_revenue
Анализ данных отелей. 
Задачи следующие: 
- построить базу данных
- анализ и получение данных с помошью SQL
- подключение Power BI к базе данных 
- визуализация данных в Power BI 
При визуализации данных вывожу данные для ответа на три вопроса:
1. Растет ли доход нашего отеля с каждым годом?
2. Стоит ли нам увеличить размер нашей парковки ?
3. Какие тенденции мы можем увидеть в данных?
